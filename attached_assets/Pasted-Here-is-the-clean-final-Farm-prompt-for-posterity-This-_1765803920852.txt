## Replit Agent 3 MAX — Farm Grader Canonical Prompt

**ROLE**
You are working on **AOS Farm**, the canonical **grader** for AutonomOS.

---

## Non-Negotiables

1. **Farm is the grader.**

   * All grading is keyed by `run_id`.
   * No grading without run provenance.

2. **Single source of truth**

   * **Supabase Postgres only**.
   * Ignore Replit DB completely.
   * No SQLite, no JSON disk storage.

3. **Deterministic, fact-based**

   * No ML.
   * No scoring curves.
   * No inferred timestamps.
   * Only observed evidence.

4. **Fail fast**

   * Ambiguous configuration → fatal startup error.
   * No silent fallbacks.

---

## Environment Contract

* Use `SUPABASE_DB_URL` if set, else `DATABASE_URL`.
* If neither exists → fail startup.
* If `IGNORE_REPLIT_DB=true`, ignore all env vars starting with `REPLIT`.
* Report selected DB provider at startup (name only, no secrets).

## Required Data Model (Minimum)

### `runs`

* `run_id` (TEXT / UUID, PK)
* `run_fingerprint` (stable hash of config + seed)
* `created_at`
* `seed`
* `schema_version`
* `enterprise_profile`
* `realism_profile`
* `scale`

### `snapshots`

* `snapshot_id` (PK)
* `run_id` (FK → runs.run_id, NOT NULL)
* `sequence` (INTEGER, default `0`)
* `snapshot_fingerprint`
* `created_at`
* `snapshot_json` (payload)
* Snapshot payload **must include `observed_at` evidence**

### (Optional, later) `grades`

* `grade_id`
* `run_id`
* `target_system`
* `submitted_at`
* `result_json` (facts only)

## Runtime Rules

1. **Run first**

   * Every Farm execution must create exactly **one** `runs` row.

2. **Snapshots belong to a run**

   * Every snapshot must reference `run_id`.
   * No orphan snapshots allowed.

3. **Static today**

   * Only `sequence = 0` is produced for now.
   * Mono-dynamic (sequence > 0) comes later.
